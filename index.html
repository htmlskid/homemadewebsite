Our Homemade Hacks

Bookmarklet 1: Open a new tab with a specific URL

javascript:window.open('https://example.com');

Bookmarklet 2: Change the page title

javascript:document.title = 'New Page Title';

Bookmarklet 3: Scroll to the bottom of the page

javascript:(function() { var lastKnownScrollPosition = 0; var ticking = false; document.documentElement.scrollingElement.addEventListener('scroll', function(e) { lastKnownScrollPosition = window.scrollY || document.documentElement.scrollTop || document.body.scrollTop; if (!ticking) { window.requestAnimationFrame(function() { ticking = false; window.scrollTo(0, lastKnownScrollPosition + document.documentElement.clientHeight); }); ticking = true; } }); })();

Bookmarklet 4: Highlight all text on the page

javascript:(function() { var range, selection; if (window.getSelection) { range = document.createRange(); selection = window.getSelection(); range.selectNodeContents(document.body); range.collapse(false); selection.removeAllRanges(); selection.addRange(range); document.body.style.backgroundColor = 'yellow'; } })();

Bookmarklet 5: Save the page as a PDF

javascript:(function() { var doc = new jsPDF('p', 'pt', 'letter'); source = document.getElementById('content'); specialElementHandlers = { '#editor': function (element, renderer) { return true; } }; margins = { top: 80, bottom: 60, left: 40, width: 522 }; doc.fromHTML(source, margins.left, margins.top, { 'width': margins.width, 'elementHandlers': specialElementHandlers }); doc.save('MyWebsite.pdf'); })();

Tutorial: Creating a JavaScript Bookmarklet

1. Open your web browser and navigate to the bookmark manager (usually located in the bookmarks bar or under the bookmarks menu).

2. Create a new bookmark and name it something like "My Bookmarklet".

3. In the URL field, paste the JavaScript code from one of the bookmarklets above.

4. Replace the comment with the desired functionality.

5. Save the bookmark. Now, when you click the bookmark, the specified JavaScript code will run on the current webpage.
